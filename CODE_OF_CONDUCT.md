# Código de conduta

Este projeto adota o [Contributor Covenant 2.1](https://www.contributor-covenant.org/pt-br/version/2/1/code_of_conduct/).

Esperamos que todos os participantes — mantenedores, contribuidores e usuários — mantenham um ambiente respeitoso e construtivo. Comportamento abusivo, discriminatório, assediador ou desonesto (por exemplo, submissão de dados inventados ou falsificação de resultados de benchmark) não será tolerado e pode resultar em banimento do projeto.

Denúncias podem ser enviadas para os mantenedores do repositório via issue privada ou e-mail listado no `SECURITY.md`.
